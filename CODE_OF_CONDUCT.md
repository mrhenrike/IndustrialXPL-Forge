# Code of Conduct — IndustrialXPL-Forge

## Our Commitment

IndustrialXPL-Forge is an offensive and defensive security research framework for OT/ICS/SCADA environments. All participants commit to:

- Using this tool **only on systems you own or have explicit written authorization to test**
- Contributing to the security community through responsible research
- Treating all contributors with respect and professionalism

## Acceptable Use

- Authorized penetration testing engagements
- Red team / blue team exercises in isolated lab environments
- Security research under coordinated disclosure policies
- Academic research in controlled environments
- Defensive preparation and incident response training

## Unacceptable Use

- Using modules against systems without explicit written authorization
- Submitting modules designed to cause unauthorized harm
- Targeting critical infrastructure without authorization
- Distributing credentials, keys, or proprietary exploit code in contributions
- Compiling or deploying incorporated malware artifacts outside isolated lab environments

## Malware Research (public corpus)

IXF **publicly ships** native modules and vendor trees for 12 incorporated malware families
(Mirai, TRISIS/TRITON, Bashlite, Akaja, and others) plus ICS-tool references for
**defensive TTP mapping and detection engineering**.

**By cloning or using this repository you accept sole responsibility** for lawful,
authorized use. See [TERMS_OF_USE.md](TERMS_OF_USE.md), [DISCLAIMER.md](DISCLAIMER.md),
and [SECURITY.md](SECURITY.md).

- Do not compile or deploy incorporated artifacts outside isolated labs without authorization
- Do not target critical infrastructure without written permission

---

## Responsible Disclosure

If you discover a vulnerability in widely deployed OT/ICS equipment:

1. Document the vulnerability in a controlled lab
2. Notify the vendor through their disclosure channel
3. Allow 90 days minimum for critical vulnerabilities
4. After disclosure, contribute a module to this repository

## Enforcement

Contributions that violate this code of conduct will be rejected.
